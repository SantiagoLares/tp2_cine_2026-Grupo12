## Descripción

Maquetado de la interfaz de la plataforma de cine **Cine Aurora**, aplicando Flexbox, Grid, Media Queries y modularización CSS.

## Páginas

- `index.html` — Cartelera principal (Grid + Media Queries)
- `detalle.html` — Detalle de película (Flexbox + Media Queries)
- `estrenos.html` — Próximos lanzamientos (Grid)
- `login.html` — Acceso de usuarios (Flexbox)
- `registro.html` — Registro de nuevos usuarios (Flexbox + Grid)
- `perfil.html` — Panel de usuario (Grid + Flexbox)

## Tecnologías

- HTML5 semántico
- CSS3 modularizado con `@import`
- Flexbox y CSS Grid
- Media Queries (responsive)
- Variables CSS (`:root`)
